Froggiesplaining :
Dans ce 7ème cours, partie B, on découvre Le "Container" et les "Media Queries".
qui permet une mise en page réactive basée sur les dimensions du conteneur parent plutôt que sur la fenêtre elle-même. Explication :

"container-type: inline-size" Active les container queries sur le conteneur parent. La taille du conteneur pourra être utilisée dans des requêtes CSS.

"@container" Similaire aux media queries mais basé sur la taille du conteneur parent plutôt que sur la taille de la fenêtre.
Par exemple :
- Si la largeur du conteneur parent est d'au moins 500px, le fond de ".box" devient vert.
- Si la largeur atteint 700px, le fond devient corail et la taille de la police augmente.

fonctionne sur chrome, edge et safari (pas sur firefox !)

Conteneur Fixe

Conteneur Fluide

Ce conteneur prend toute la largeur de l'écran, indépendamment de la taille de la fenêtre.

Conteneur MD

Ce conteneur est pour les écrans moyens (≥768px).

Conteneur LG

Ce conteneur est pour les grands écrans (≥992px).

Conteneur XL

Ce conteneur est pour les très grands écrans (≥1200px).

Conteneur XXL

Ce conteneur est pour les écrans ultra larges (≥1400px).

Dans cette 3ème leçon, on découvre Flex:

Flex Container Pour activer le mode Flexbox sur un conteneur, on utilise la propriété CSS display: flex ou display: inline-flex. Cela transforme le conteneur en un conteneur flexible et les éléments à l'intérieur deviennent des "flex items"




Justify Content La propriété justify-content contrôle l’alignement des éléments le long de l'axe principal (généralement l’axe horizontal si les éléments sont en ligne).
Valeurs communes de justify-content :
flex-start : Aligne les éléments au début du conteneur.
flex-end : Aligne les éléments à la fin du conteneur.
center : Centre les éléments.
space-between : Distribue les éléments avec un espace égal entre eux, le premier élément est collé au bord gauche et le dernier au bord droit.
space-around : Distribue les éléments avec des espaces égaux autour de chaque élément, incluant les bords du conteneur.
space-evenly : Distribue les éléments avec un espace égal entre tous, y compris les bords du conteneur.

Bootstrap est un framework CSS gratuit et open-source qui facilite la création de sites web réactifs (compatibles sur mobile, tablette et desktop) avec un design moderne. Il fournit une collection de classes prêtes à l'emploi pour la mise en page, la typographie, les boutons, les formulaires, les tableaux, etc.

Avantages pour les débutants :
- Facilité d'utilisation : Il suffit d'inclure le fichier CSS/JS et d'utiliser les classes prédéfinies.
- Design réactif : Sites automatiquement adaptés aux différentes tailles d'écrans sans effort supplémentaire.
- Gain de temps : Des composants déjà stylés (menus, cartes, carrousels, etc.) permettent de créer des interfaces rapidement.
- Standardisation : Le design est cohérent et professionnel, même sans être expert en CSS.

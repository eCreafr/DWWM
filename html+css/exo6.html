Froggiesplaining :


inline : L'élément est affiché dans le flux en ligne, sans rupture de ligne.
block : L'élément occupe toute la largeur disponible et commence sur une nouvelle ligne.
inline-block : L'élément est en ligne mais accepte des propriétés de dimensionnement comme un élément bloc. (il ne prend pas toute la ligne)
flex : Utilise un conteneur flexbox pour disposer ses éléments enfants de manière flexible.
inline-flex : Combine flex avec le comportement inline.
grid : Définit un conteneur de mise en page en grille (nouveau en HTML5).
none : Cache l'élément.



Avec le framework Bootstrap :
il suffira d'ecrire :
La classe d-inline permet de rendre un élément inline.
La classe d-inline-block permet de rendre un élément inline-block.
La classe d-block force un élément à se comporter comme un block.
La classe d-flex active un conteneur flexbox.
La classe d-inline-flex combi ne le comportement inline avec flexbox.
grid : Définit un conteneur de mise en page en grille (nouveau en HTML5).
La classe d-none masque l'élément.

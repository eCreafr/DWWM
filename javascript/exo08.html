Compteur : 0

Froggiesplaining : Explications détaillées :

  • let counter = 0;Initialise la variable counter à 0.
  • Sélection des éléments : getElementById() permet de lier le JavaScript aux éléments HTML du DOM.
  • addEventListener("click", ...)Ajoute un gestionnaire d'événements pour que le compteur s'incrémente à chaque clic
  • counterDisplay.textContent = `Compteur : ${counter}`;Mise à jour de l'affichage : Modifie le contenu texte de l'élément pour afficher la valeur actuelle du compteur.

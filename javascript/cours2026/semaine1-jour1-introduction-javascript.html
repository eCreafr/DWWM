🚀 Semaine 1 - Jour 1 : Introduction à JavaScript

📚 Qu'est-ce que JavaScript ?

JavaScript est le langage de programmation du web. Il permet de rendre les pages web interactives et dynamiques.

Exemples d'utilisation : valider des formulaires, créer des animations, charger du contenu sans recharger la page, créer des jeux, des applications web complètes...

Important : JavaScript est différent de Java ! Ce sont deux langages distincts.

1. Où écrire du JavaScript ?

3 méthodes principales :

  • Dans la balise <script> (dans le HTML)
  • Dans un fichier externe (script.js)
  • Dans la console du navigateur (F12)
<!-- Dans le HTML -->
<script>
  console.log('Bonjour !');
</script>

2. console.log()

La commande console.log() permet d'afficher des messages dans la console du navigateur.

Très utile pour :

  • Vérifier si le code fonctionne
  • Afficher des valeurs
  • Déboguer (trouver les erreurs)
console.log('Bonjour le monde !');
console.log(42);
console.log(2 + 2);

3. Commentaires

Les commentaires permettent d'expliquer le code. Ils ne sont pas exécutés.

// Commentaire sur une seule ligne

/*
  Commentaire
  sur plusieurs
  lignes
*/

✅ Bonne pratique : Commenter son code pour le rendre plus lisible !

4. Instructions et point-virgule

Chaque instruction JavaScript se termine par un point-virgule (;) (optionnel mais recommandé).

console.log('Première ligne');
console.log('Deuxième ligne');

💡 Astuce : Même si JavaScript ajoute automatiquement les points-virgules, il est préférable de les écrire explicitement.

# 📚 Formation JavaScript - Plan de cours sur 4 semaines

## 🎯 Objectif
Formation complète JavaScript pour adultes débutants, du niveau zéro jusqu'à la maîtrise du DOM, de l'asynchrone et de la lecture/adaptation de code JavaScript moderne.

## 📋 Structure du cours

### **Semaine 1 : Fondamentaux JavaScript & Syntaxe de base** ✅ COMPLET
1. [Introduction à JavaScript](semaine1-jour1-introduction-javascript.html) - Console, scripts, premiers pas
2. [Variables et types de données](semaine1-jour2-variables-types.html) - let, const, string, number, boolean
3. [Opérateurs et expressions](semaine1-jour3-operateurs-expressions.html) - Arithmétiques, comparaison, logiques, template literals
4. [Structures conditionnelles](semaine1-jour4-structures-conditionnelles.html) - if/else, switch, ternaire
5. [Boucles](semaine1-jour5-boucles.html) - for, while, break, continue

**📦 Projet Semaine 1 :** Programme qui affiche la table de multiplication d'un nombre

---

### **Semaine 2 : Fonctions, Tableaux & Objets** ✅ COMPLET
1. [Fonctions](semaine2-jour1-fonctions.html) - Déclaration, paramètres, return, fonctions fléchées, scope
2. [Tableaux - Partie 1](semaine2-jour2-tableaux-base.html) - Création, accès, push, pop, shift, unshift, parcours
3. [Tableaux - Partie 2](semaine2-jour3-tableaux-methodes.html) - map, filter, reduce, find, sort, slice, splice
4. [Objets](semaine2-jour4-objets.html) - Création, propriétés, méthodes, this
5. [Tableaux d'objets](semaine2-jour5-tableaux-objets.html) - Manipulation complexe, destructuration

**📦 Projet Semaine 2 :** Gestionnaire de tâches (To-Do List) en console avec tableaux d'objets

---

### **Semaine 3 : DOM & Interaction avec la page** ✅ COMPLET
1. [Introduction au DOM](semaine3-jour1-introduction-dom.html) - Sélecteurs, textContent, innerHTML
2. [Manipulation du DOM](semaine3-jour2-manipulation-dom.html) - Attributs, styles, classes, création d'éléments
3. [Événements](semaine3-jour3-evenements.html) - addEventListener, click, input, submit, event object
4. [Formulaires & Validation](semaine3-jour4-formulaires-validation.html) - Récupération valeurs, validation côté client
5. [Projet DOM interactif](semaine3-jour5-projet-dom.html) - Quiz interactif avec score en temps réel

**📦 Projet Semaine 3 :** To-Do List interactive avec interface graphique (HTML + JS)

---

### **Semaine 4 : Concepts avancés & Asynchrone** ✅ COMPLET
1. [Asynchronisme - Introduction](semaine4-jour1-asynchronisme-intro.html) - setTimeout, setInterval, callbacks
2. [Promises & Fetch API](semaine4-jour2-promises-fetch.html) - Promises, then/catch, fetch, JSON
3. [Async/Await](semaine4-jour3-async-await.html) - Syntaxe moderne, try/catch, gestion d'erreurs
4. [Modules & Bonnes pratiques](semaine4-jour4-modules-bonnes-pratiques.html) - Import/export, spread operator, organisation code
5. [Lecture et adaptation de code](semaine4-jour5-lecture-code.html) - Déboguer, comprendre patterns, adapter code existant

**📦 Projet Final Semaine 4 :**
- Application météo complète
- Récupération données via API (OpenWeatherMap)
- Interface interactive
- Gestion erreurs
- LocalStorage pour sauvegarder favoris
- Code organisé en modules

## 🚀 Comment utiliser ce cours

### 1. Commencez par l'index
Ouvrez [index.html](index.html) dans votre navigateur pour une vue d'ensemble interactive.

### 2. Suivez l'ordre recommandé
Les cours sont conçus pour être suivis dans l'ordre, chaque leçon s'appuyant sur les précédentes.

### 3. Pratiquez avec les exercices
Chaque cours contient une **console JavaScript interactive** pour tester le code en direct. **Ne passez pas à la leçon suivante sans avoir pratiqué !**

### 4. Utilisez les DevTools
Apprenez à utiliser la console du navigateur (F12) pour déboguer et expérimenter.

### 5. Consultez la documentation
- [MDN Web Docs - JavaScript](https://developer.mozilla.org/fr/docs/Web/JavaScript) - Référence complète
- [JavaScript.info](https://javascript.info/) - Tutoriels détaillés
- [DevDocs](https://devdocs.io/) - Documentation rapide
- [Can I Use](https://caniuse.com/) - Compatibilité navigateurs

## 💡 Conseils pour réussir

### ✅ À faire
- **Pratiquer tous les jours** (même 30 minutes)
- **Coder en même temps** que vous suivez le cours
- **Expérimenter** : modifier les exemples, tester des variations
- **Utiliser la console interactive** intégrée dans chaque cours
- **Déboguer avec console.log()** : afficher les valeurs pour comprendre
- **Lire les messages d'erreur** : ils indiquent souvent la solution
- **Poser des questions** : rechercher sur Google, Stack Overflow, MDN

### ❌ À éviter
- Lire passivement sans coder
- Copier-coller sans comprendre
- Sauter les exercices
- Vouloir aller trop vite
- Se décourager face aux erreurs (c'est normal et nécessaire !)
- Négliger la console du navigateur (F12)

## 🎯 Roadmap d'apprentissage

### Phase 1 : Fondations (Semaines 1-2)
- Maîtriser la syntaxe JavaScript de base
- Comprendre les structures de données (tableaux, objets)
- Créer des fonctions réutilisables
- Manipuler des données complexes

### Phase 2 : Interaction web (Semaine 3)
- Manipuler le DOM
- Gérer les événements utilisateur
- Créer des interfaces interactives
- Valider des formulaires

### Phase 3 : Concepts avancés (Semaine 4)
- Maîtriser l'asynchronisme
- Récupérer des données d'APIs
- Organiser son code professionnellement
- Lire et adapter du code existant

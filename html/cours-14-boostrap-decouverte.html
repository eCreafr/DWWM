🎯 Objectifs de cet exercice :
Découvrir Bootstrap, le framework CSS le plus populaire au monde pour créer des sites web responsive rapidement.

📦 Qu'est-ce que Bootstrap ?
Bootstrap est un framework CSS gratuit et open-source qui facilite la création de sites web réactifs (compatibles sur mobile, tablette et desktop) avec un design moderne. Il fournit une collection de classes prêtes à l'emploi pour la mise en page, la typographie, les boutons, les formulaires, les tableaux, etc.

✨ Avantages pour les débutants :

  • Facilité d'utilisation : Il suffit d'inclure le fichier CSS/JS via CDN et d'utiliser les classes prédéfinies
  • Design réactif automatique : Sites automatiquement adaptés aux différentes tailles d'écrans (mobile-first)
  • Gain de temps : Des composants déjà stylés (navbar, cards, carousels, modals...) prêts à l'emploi
  • Standardisation : Design cohérent et professionnel, même sans être expert en CSS
  • Système de grille responsive : 12 colonnes flexibles pour créer n'importe quelle mise en page

🔧 Concepts clés dans cet exercice :
  • .container et .container-fluid : Conteneurs responsive
  • .row : Ligne de grille (contient les colonnes)
  • .col-* : Colonnes responsive (col-12 pour mobile, col-md-4 pour desktop...)
  • Classes utilitaires : p-4 (padding), my-4 (margin vertical), text-center, etc.
  • Composants : .nav, .list-group, etc.

📚 Pour aller plus loin : Consultez la documentation officielle Bootstrap !

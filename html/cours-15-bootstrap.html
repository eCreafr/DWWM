Conteneur Fixe

Ceci est un conteneur qui s'ajuste selon des points de rupture (breakpoints).

Conteneur Fluide

Ce conteneur prend toute la largeur de l'écran, indépendamment de la taille de la fenêtre.

Conteneur SM

Ce conteneur est pour les petits écrans (≥576px).

Conteneur MD

Ce conteneur est pour les écrans moyens (≥768px).

Conteneur LG

Ce conteneur est pour les grands écrans (≥992px).

Conteneur XL

Ce conteneur est pour les très grands écrans (≥1200px).

Conteneur XXL

Ce conteneur est pour les écrans ultra larges (≥1400px).

🎯 Objectifs de cet exercice :
Comprendre les différents types de conteneurs Bootstrap et leurs breakpoints (points de rupture).

📦 Les conteneurs Bootstrap :

  • .container : Conteneur responsive avec largeur maximale selon l'écran (540px/720px/960px/1140px/1320px)
  • .container-fluid : Prend toujours 100% de la largeur disponible
  • .container-{breakpoint} : Conteneurs qui deviennent fixes à partir d'un certain breakpoint

📏 Les breakpoints Bootstrap :
  • xs (extra small) : < 576px
  • sm (small) : ≥ 576px
  • md (medium) : ≥ 768px
  • lg (large) : ≥ 992px
  • xl (extra large) : ≥ 1200px
  • xxl (extra extra large) : ≥ 1400px

💡 Astuce : Redimensionnez votre navigateur pour voir comment chaque conteneur réagit différemment aux changements de taille !

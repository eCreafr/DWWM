🎯 Objectifs de cet exercice :
Dans ce premier cours, vous allez découvrir la structure de base d'un document HTML.

📋 Les balises HTML essentielles :

  • <!DOCTYPE html> : Indique au navigateur qu'il s'agit d'un document HTML5
  • <html lang="fr"> : Balise racine qui englobe tout le document
  • <head> : En-tête du document (métadonnées invisibles)
    • charset UTF-8 : Permet d'afficher correctement les accents et caractères spéciaux
    • <title> : Titre affiché dans l'onglet du navigateur
    • viewport : Configuration pour un affichage responsive (adapté aux mobiles)
    • <link rel="stylesheet"> : Lien vers la feuille de style CSS
  • <body> : Corps du document (contenu visible)
    • <header> : En-tête de page
    • <nav> : Menu de navigation
    • <main> : Contenu principal
    • <section> : Section thématique
    • <article> : Contenu autonome
    • <footer> : Pied de page

  • Les balises de titre <h1> à <h6> (hiérarchie des titres)
  • Comment intégrer une image avec <img src="..." alt="...">
  • Créer des paragraphes avec <p>
  • Mettre du texte en gras (<strong>) et en italique (<em>)
  • Créer des liens hypertextes avec <a href="...">

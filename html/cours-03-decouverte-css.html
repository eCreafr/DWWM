Froggiesplaining :


🎯 Objectifs de cet exercice :
Dans ce 2ème cours, vous allez découvrir les bases du CSS et du JavaScript pour rendre vos pages plus interactives et visuellement attractives.

🎨 Techniques CSS que vous allez apprendre :

  • Les dégradés CSS (gradients) : créer des transitions de couleurs fluides avec linear-gradient()
  • Les box-shadow : ajouter des ombres portées autour des éléments pour créer de la profondeur
  • Images en arrière-plan : utiliser background-image pour styliser vos conteneurs
  • Animations CSS simples : créer des effets visuels avec @keyframes et animation
  • Effets hover : changer l'apparence des éléments au survol de la souris avec :hover
  • Z-index : gérer la superposition des éléments (quel élément apparaît au-dessus d'un autre)

⚡ JavaScript - Interactivité :
  • Formulaire HTML + JavaScript : capturer les données saisies et les afficher dynamiquement
  • addEventListener : écouter les événements utilisateur (clic, survol, soumission...)
  • Changer une image au survol : deux méthodes sont présentées dans cette page
    • En CSS : changement de taille (transform: scale)
    • En JavaScript : changement de la source de l'image (img.src)

💡 Conseil : Ouvrez le code source dans Visual Studio Code et essayez de modifier les couleurs, les tailles d'ombres ou les animations pour voir les effets en temps réel !

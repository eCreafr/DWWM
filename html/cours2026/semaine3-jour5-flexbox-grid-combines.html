🎯 Semaine 3 - Jour 5 : Flexbox + Grid Combinés

📚 La puissance du combo

Flexbox et Grid ne sont pas concurrents, ils sont complémentaires ! Grid pour la structure globale, Flexbox pour les composants. Découvrons comment les combiner efficacement.

⚔️ Flexbox vs Grid : Quand utiliser quoi ?

Critère 🔵 Flexbox 🟢 Grid
Dimensions 1D (ligne OU colonne) 2D (lignes ET colonnes)
Usage idéal Composants, navigation, alignement Layouts de page, grilles
Exemples Navbar, cards internes, boutons, centrage Page layout, galeries, dashboards
Contrôle Content-out (du contenu vers le layout) Layout-in (du layout vers le contenu)
Flexibilité Items s'adaptent dynamiquement Structure définie à l'avance

🏗️ Exemple 1 : Layout complet (Grid + Flex)

Structure de page : Grid pour le layout, Flex pour les composants

/* Grid pour la structure globale */
.layout {
  display: grid;
  grid-template-areas:
    "header header"
    "sidebar main"
    "footer footer";
}

/* Flex pour le header */
.header {
  display: flex;
  justify-content: space-between;
  align-items: center;
}

🎴 Exemple 2 : Grille de Cards

Grid pour la grille, Flex à l'intérieur des cards

/* Grid pour disposer les cards */
.card-grid {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(280px, 1fr));
}

/* Flex à l'intérieur des cards */
.card-body {
  display: flex;
  flex-direction: column;
}

.card-actions {
  display: flex;
  gap: 10px;
}

💡 Best Practices

✅ Utiliser Grid pour

  • Layout de page (header, sidebar, main, footer)
  • Grilles de produits / articles
  • Dashboards avec widgets
  • Galeries d'images
  • Structures 2D complexes

✅ Utiliser Flexbox pour

  • Navigation (navbar)
  • Contenu interne des cards
  • Alignement vertical/horizontal
  • Groupes de boutons
  • Composants 1D

🎯 La règle d'or

Grid = Structure globale (le squelette)
Flexbox = Composants et alignement (les muscles)

💪 Ensemble, ils créent des layouts puissants et maintenables !

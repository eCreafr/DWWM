Les iFrames

Froggiesplaining :


🎯 Objectifs de cet exercice :
Découvrir la balise <iframe> (inline frame) pour intégrer du contenu externe dans une page HTML.

📦 Qu'est-ce qu'une iframe ?
Une iframe permet d'"encapsuler" un site ou contenu externe directement dans votre page web. C'est comme une fenêtre vers un autre site web.

💡 Usages courants :

  • Google Maps : Intégrer une carte interactive
  • Vidéos YouTube : Intégrer une vidéo sans héberger le fichier
  • Réseaux sociaux : Intégrer des publications Facebook, Twitter...
  • Formulaires externes : Intégrer des formulaires Google Forms, Typeform...

🔧 Attributs importants :
  • src : URL du contenu à afficher dans l'iframe
  • allowfullscreen : Autorise le mode plein écran
  • loading="lazy" : Charge l'iframe uniquement quand elle devient visible (améliore les performances)
  • width et height : Définissent les dimensions (peuvent être définies en CSS)

⚠️ Attention : Tous les sites n'autorisent pas l'intégration en iframe pour des raisons de sécurité.
